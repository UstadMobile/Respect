# Invite new users using QR code or link

## Summary
- Admin generates invite link (QR/link) for teacher
- Teacher joins using QR/link → creates account
- Teacher creates class and generates invite code for student
- Student joins using invite code → waits for approval
- Teacher approves student → student joins class
- Teacher generates parent invite link to join class
- Parent joins using link → adds child to class

---

## Flow A: Admin invites teacher via link/QR
- Admin logs in
- Navigates to **People**
- Clicks **Invite person**
- Selects role as **Teacher**
- Disables approval (optional)
- Copies invite link

---

## Flow B: Teacher joins using QR/link
- Launch app → Get Started
- Scans QR / pastes invite URL
- Verifies invitation details (role, school)
- Accepts terms
- Completes profile
- Creates account
- Lands on **Apps** screen

---

## Flow C: Teacher creates class & invites student
- Teacher creates new class
- Opens class details
- Clicks **Add Student → Invite person**
- Copies invite code

---

## Flow D: Student joins using invite code
- Launch app → select school
- Click **I have an invite code**
- Enters invite code
- Verifies class details
- Completes profile
- Creates account
- Lands on **Waiting for approval**

---

## Flow E: Teacher approves student
- Teacher logs in
- Opens class
- Views pending requests
- Approves student

---

## Flow F: Teacher invites parent via link to join class 
- Opens class
- Clicks **Add student → Invite person → Invite via parents**
- Disables approval 
- Copies invite link

---

## Flow G: Parent joins using invite link and adds child to class
- Opens invite link
- Launches app → Get Started
- Verifies invitation details
- Accepts terms
- Completes parent profile
- Creates account
- Adds child profile
- Completes signup
