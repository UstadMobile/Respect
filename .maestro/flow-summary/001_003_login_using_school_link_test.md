# Login using school link

## Summary
- User opens app via school link
- Validates empty and invalid link inputs
- Enters valid school URL
- Logs in using credentials
- Accesses app successfully

---

## Flow A: Launch app with school link option
- Launch app with school URL
- Tap **Get Started**
- Select **Other options**

---

## Flow B: Validate link input
- Tap on link field → click **Next**
- Verify error for empty input
- Enter invalid text → click **Next**
- Verify invalid URL error

---

## Flow C: Enter valid school link
- Clear input field
- Enter valid school URL
- Tap **Next**
- User directed to Login screen

---

## Flow D: User login
- Enter username and password
- Tap **Login**
- Handle optional save password prompt
- Verify user lands on **Apps** screen
