# Add school via self-registration

## Summary
- User adds a new school from login screen
- Selects host and registers school
- Creates system administrator account
- Logs into newly created school
- Verifies profile and logout/login flow

---

## Flow A: Start school self-registration
- Launch app
- Tap **Get Started**
- Enter school name
- Select **Add my school**

---

## Flow B: Select host and register school
- Select school host URL
- Verify **Register New School** screen
- Enter school name and school URL
- Tap **Next**

---

## Flow C: Verify invitation details
- Verify role as **System administrator**
- Verify school name and URL
- Proceed to next step

---

## Flow D: Accept terms and create profile
- Accept **Terms and conditions**
- Enter profile details (name, gender, DOB)
- Proceed to account creation

---

## Flow E: Create admin account
- Verify username generation
- Enter password
- Complete sign-up
- Verify navigation to **Apps** screen

---

## Flow F: Verify profile and logout
- Open profile from account icon
- Verify username and school URL
- Logout from account

---

## Flow G: Login to registered school
- Select newly created school
- Enter admin credentials
- Login successfully
- Verify **Apps** screen
