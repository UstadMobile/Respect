# Admin assigns assignment to a class

## Summary
- Admin setup includes app, class and teacher creation
- Teacher logs in and accesses class
- Teacher creates a new assignment
- Assignment is linked with lesson content
- Assignment is saved and verified in class

---

## Flow A: Admin add App, Class and Teacher on Respect
- Admin logs in to app
- Adds new app
- Create a class
- Create teacher account 
- Add teacher to class

---

## Flow B: Teacher login
- Launch app
- Select school
- Login with teacher credentials
- Verify access to **Apps** screen

---

## Flow C: Navigate to class and assignments
- Open **Classes**
- Open **Assignments** section
- Start creating new assignment

---

## Flow D: Create assignment
- Click **Add Assignment**
- Validate required fields
- Enter assignment name
- Select class
- Set due date and time

---

## Flow E: Link lesson content
- Select **Lesson/Assessment**
- Open app content
- Choose Grade 1 lesson
- Select specific lesson file
- Save assignment

---

## Flow F: Verify assignment creation
- Confirm assignment appears in class
- Validate linked lesson content
