#  Admin create a person and validate user login and credentials

## Description:

Admin create a person and validate user login and credentials

## Step-by-step procedure:

### A) Admin add new Person

1. Launch the application.
2. Click **Get started** button on On-boarding screen.
3. On "Let's get started" screen, Tap on the **School name** field.
4. Enter the **School name**.
5. Select on School name that will appear below the School name field.
6. Verify the user is in **Login** screen.
7. Tap on the **Username** field and enter the admin username.
8. Tap on the **Password** field and enter the admin password.
9. Tap on the **Login** button.
10. Verify the user is in the **Apps** screen.
11. Click on the "People" tab.
12. Click on "+ Person" button.
13. On "Add person" screen, click on Save button.
14. Verify that user getting error saying First name, Last name and Gender fields are required fields.
15. Click on "First name" field and enter the user's First name.
16. Click on "Last name" field and enter the user's Last name.
17. Click on "Gender" and select user's gender.
18. Click on "Date of birth" field and enter a future date.
19. Click on the Save button.
20. Verify user is getting error saying- Date of birth can't be a future date.
21. Click on "Date of birth" field and enter user's date of birth.
22. Click on the Save button
23. Verify person details are saved.
24. On Person details screen click on "Edit" button.
25. On "Edit person" screen, click on country code section and choose the country code as "+971"
26. Click on Phone number field and enter a single digit number.
27. Click on the Save button.
28. Verify user is getting error saying- Invalid phone number
29. Click on Phone number field and enter 11 digit number
30. Click on the Save button.
31. Verify user is getting error saying- Invalid phone number.
32. Click on Phone number field and enter 10 digit number.
33. Click on email field and enter user's name.
34. Click on the Save button.
35. Verify user is getting error saying- Invalid email id.
36. Click on email field and enter user's valid email id.
37. Click on the Save button
38. Verify user details are saved.
39. Click on "Create account" button.
40. Verify the user able to see username in Create Account screen. 
41. Verify "Assign QR code badge" and "Set password" options are visible under the username section. 
42. Click on "Set password".
43. New screen appears with "Password*" field, click on "Save".
44. Verify user getting "Required field" error. 
45. Click on "Password*" field and enter new password then click on Save 
46. Verify user is on "Manage account" screen. 
47. Click on "Assign QR code badge". 
48. Verify "Scan QR code badge" screen appears. 
49. Click on "More options" and click on Paste URL button and enter the invalid QR Badge URL/Code. 
50. Click "OK" and verify user getting error saying - Invalid url
51. Enter valid url and click on "Ok".
52. Verify user on Manage account screen with QR code badge and Change button next to it.
53. Click on "Change" button next to QR code badge. 
54. Verify 2 option to "Assign new badge (replace)" or "Revoke badge" appears.
55. Click on "Assign new badge (replace)" and verify user is taken to "Scan QR code badge" screen.
56. Go back to Manage accounts screen.
57. Click on "Change" button next to QR code badge.
58. Click on "Revoke badge" button.
59. Verify "QR code badge" is removed and "Assign QR code badge" is visible there.
60. Click on "Assign QR code badge" button. 
61. Click on "More options" and click on Paste URL button and enter the Valid QR Badge URL/Code and Click on "Ok". 
62. Verify user on Manage account screen with QR code badge and Change button next to it.

### B) New person login to the app Scanning QR badge

1. Launch the application.
2. Click **Get started** button on On-boarding screen.
3. On "Let's get started" screen, Click on the "Scan QR/badge" button.
4. Verify the "Scan QR/badge" screen is visible. 
5. Click on "More options" and click on Paste URL button.
6. Enter the QR Badge URL assigned here. 
7. Click "OK". 
8. Verify the user is successfully logged in and sees the Apps screen.
9. Click on the "People" tab.
10. Click on user's name 
11. Verify user is on Person details screen.
12. Click on "Manage account" button.
13. Verify user able to see username and password change button is visible.
14. Click on "Change" button next to the password button.
15. Click on "Old password" and enter old password.
16. Click on "New password" and enter new password.
17. Click on "Save" button.
18. Verify new password got saved.

### C) New person login to the app with new password

1. Launch the application.
2. Click **Get started** button on On-boarding screen.
3. On "Let's get started" screen, Tap on the **School name** field.
4. Enter the **School name**.
5. Select on School name that will appear below the School name field.
6. Verify the user is in **Login** screen.
7. Tap on the **Username** field and enter the valid username.
8. Tap on the **Password** field and enter the updated new password.
9. Tap on the **Login** button.
10. Verify the user is in the **Apps** screen.
