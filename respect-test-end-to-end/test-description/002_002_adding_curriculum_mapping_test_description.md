# Admin adds mapping by linking lessons from integrated edtech apps

## Description:
An admin adds a new curriculum mapping by creating a book, adding chapters and lessons from an integrated edtech app.
## Step-by-step procedure:

1. Launch the application.
2. Click **Get started** button on On-boarding screen.
3. On "Let's get started" screen, Tap on the **School name** field.
4. Enter the **School name**.
5. Select on School name that will appear below the School name field.
6. Verify the user is in **Login** screen.
7. Tap on the **Username** field and enter the admin username.
8. Tap on the **Password** field and enter the admin password.
9. Tap on the **Login** button.
10. Verify the user is in the **Apps** screen.
11. Tap on the button "+ App" to add a new app.
12. Select the option “Add from Link”.
13. Tap on the input field labelled “Link”.
14. Enter the URL of Edtech app.
15. Tap “Next” to proceed.
16. Verify that the app detail screen is displayed with title “App detail”.
17. Verify the app name is visible.
18. Verify buttons “Try it”, “Add App”, and “Lessons” are visible.
19. Click on "Add App" button.
20. Go to Apps screen and verify the App got added to Apps screen.
21. Tap on Settings.
22. Select Mapping.
23. Verify the Mapping screen is displayed with the title “Mapping”.
24. Tap on the + Map button.
25. Verify the screen title is “Mapping edit”.
26. Tap on Save without entering details.
27. Verify the validation message “Required field” is displayed.
28. Tap on Book title* and enter a title.
29. Tap on Book description and enter a description.
30. Tap on + Chapter.
31. Verify Chapter 1 is displayed.
32. Tap on + Lesson.
33. Verify the Select lesson screen appears.
34. Confirm Browse by apps is visible.
35. Tap on My app.
36. Tap on Grade 1.
37. Tap on Lesson 1.
38. Tap on Save.
39. Verify the user is navigated back to the Mapping screen.
40. Tap on Curriculum name.
41. Verify the screen title is “Mapping edit”.
42. Tap on Add book cover and select an image.
43. Tap on Save.
44. Verify the image is displayed.
45. Verify the curriculum title is visible under Mapping list.
