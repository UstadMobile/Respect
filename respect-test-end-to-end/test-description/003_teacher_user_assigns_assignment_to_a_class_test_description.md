# Teacher Assigns a Learning Unit as Homework via Assignments Tab

## Description:
Teacher Assigns a Learning Unit as Homework via Assignments Tab

## Step-by-step procedure:

### A) Admin add new class
1. Launch the application.
2. Click **Get started** button on On-boarding screen.
3. On "Let's get started" screen, Tap on the **School name** field.
4. Enter the **School name**.
5. Select on School name that will appear below the School name field.
6. Verify the user is in **Login** screen.
7. Tap on the **Username** field and enter the username.
8. Tap on the **Password** field and enter the password.
9. Tap on the **Login** button.
10. Verify the user is in the **Apps** screen.
11. Click on the "Classes" tab.
12. Click on "+ Class" button.
13. On "Add class" screen, click on Save button.
14. Verify that user getting error saying Class name field is a required field.
15. Click on "Class name" field and enter the class name.
16. Click on "Description" field and enter the class description.
17. Click on the Save button.
18. Verify user is in newly added class detail screen.

### B.) Admin add a new app to Apps screen

1. Tap on the **Apps** tab.
2. Tap on the button "+ App" to add a new app.
3. Select the option “Add from Link”.
4. Tap on the input field labelled “Link”.
5. Enter the URL of Edtech app.
6. Tap “Next” to proceed.
7. Verify that the app detail screen is displayed with title “App detail”.
8. Verify the app name is visible.
9. Verify buttons “Try it”, “Add App”, and “Lessons” are visible.
10. Click on "Add App" button.

### C) Admin add teacher to the class

1. Admin click on "Classes" tab.
2. Admin click on the class name
3. Admin user on class detail screen clicks on "+ Add Teacher" button.
4. Admin clicks on "Add person" button.
5. Admin enters person details and select role as teacher
6. Click on Save button
7. Verify user is in "Class detail" screen.
8. Clicks on Teacher's name
9. On person detail screen, clicks on "Create account" button.
10. Enter user's username and password and click on Save.
11. Verify admin in person detail screen

### D) Admin add student A to the class

1. Admin click on "Classes" tab.
2. Admin click on the class name
3. Admin user on class detail screen clicks on "+ Add Student" button.
4. Admin clicks on "Add person" button.
5. Admin enters person details and select role as teacher
6. Click on Save button
7. Verify user is in "Class detail" screen.
8. Clicks on student's name
9. On person detail screen, clicks on "Create account" button.
10. Enter user's username and password and click on Save.
11. Verify admin in person detail screen

### E) Admin add another student to the class - follow steps in section- D

### F) Teacher Assigns a Learning Unit as Homework via Assignments Tab

1. Launch the app.
2. Click **Get started** button on On-boarding screen.
3. On "Let's get started" screen, Tap on the **School name** field.
4. Enter the **School name**.
5. Select on School name that will appear below the School name field.
6. Verify the user is in **Login** screen.
7. Tap on the **Username** field and enter the username.
8. Tap on the **Password** field and enter the password.
9. Tap on the **Login** button.
10. Verify the user is in the **Apps** screen.
11. Teacher user clicks on "Assignments" tab.
12. User clicks on "+ Assignment" button.
13. On "Add assignment" screen user clicks on "Save" button.
14. Verify "Name" and "Class" fields shows error saying- "Required field".
15. User enter a assignment name in "Name" .
16. User clicks on "Class" field and choose class from drop down list.
17. User clicks on "Date" and enter Assignment date.
18. User clicks on "Time" and choose time.
19. User clicks on "+ Lesson/assessment" button.
20. Verify user directed to "Apps" screen.
21. User clicks on a App
22. User clicks on a Module.
23. User clicks on a lesson.
24. Verify user directed to "Add assignment" screen.
25. Verify user able to see added lesson under "Assignment tasks".
26. User clicks on "Save" button.
27. Verify the Assignment got saved.
28. Verify assignment name in title, and lesson name under Assignment tasks.
29. Click on lesson name and verify user directed to specific lesson.
30. Verify user able to see Open, Download and Assign buttons.

4. Enter the **School name**.

6. Verify the user is in **Login** screen.
7. Tap on the **Username** field and enter the username.
8. Tap on the **Password** field and enter the password.
9. Tap on the **Login** button.
10. Verify the user is in the **Apps** screen.
11. Student user clicks on "Assignments" tab.
12. User clicks on "Homework 1"
13. Student clicks on the lesson and open the content.
14. User close the content.
