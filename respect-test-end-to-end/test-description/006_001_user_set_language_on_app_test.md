# Admin – Set and Change App Language

## Description:

To verify that the admin user can select language during onboarding and change the language later from Settings.

## Step-by-step procedure:

1. Launch the application.
2. Verify the language selection screen is displayed.
3. Verify the following options are visible:
4. Choose your language
5. Use system default (English)
6. हिन्दी
7. Select English.
8. Verify English is selected.
9. Tap on Get Started.
10. On school selection screen, enter the School name.
11. Select the school from the list.
12. Verify the Login screen is displayed.
13. Enter Username as admin.
14. Enter the Password.
15. Tap on Login.
16. Verify the user lands on the Apps screen.
17. Tap on Settings.
18. Verify the Settings screen is displayed.
19. Verify the current language shows as English.
20. Tap on Language.
21. Verify the following options are visible:
22. Use system default (English)
23. English
24. हिन्दी
25. Select हिन्दी.
26. Verify the screen title changes to सेटिंग्स (Settings in Hindi).
27. Tap on भाषा (Language).
28. Select English.
29. Verify the screen title changes back to Settings.
