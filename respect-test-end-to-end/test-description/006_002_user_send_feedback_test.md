# verify that a student user can submit feedback

## Description:
Teacher Assigns a Learning Unit as Homework via Assignments Tab

## Step-by-step procedure:


1. Login as School Admin.
2. Create a new Student account with valid details.
3. Logout and login using the newly created student credentials.
4. Verify the user lands on the Apps screen.
5. Tap on the User Account icon.
6. Select Share feedback.
7. Verify the Share feedback screen is displayed.
8. Select Respect launcher related issues.
9. Verify Integrated Apps related issues option is visible.
10. Select Other under Question.
11. Enter feedback text in Your feedback field.
12. Enable Airplane mode.
13. Tap Submit feedback.
14. Verify error message: “Error: try again” is displayed.
15. Disable Airplane mode.
16. Select I want a response from the Respect team.
17. Tap Submit feedback without entering contact details.
18. Verify validation message:
19. “If you need response, one of the field is required.”
20. Enter an invalid phone number.
21. Tap Submit feedback.
22. Verify Invalid error message is displayed.
23. Correct the phone number with valid format.
24. Enter an invalid email address.
25. Tap Submit feedback.
26. Verify error message: “Enter valid email address.”
27. Enter a valid email address.
28. Tap Submit feedback.
29. Verify Feedback submitted screen is displayed.
30. Verify confirmation message:
31. “Thank you for your feedback. Respect team will review it internally.”
